wwdisp.exe solves the linear water wave dispersion relation. 



wwdisp.f90 - Fortran source code

wwin.txt - example input file

wwout.txt - example output file

linux - folder for linux build inlcuding linux executable
windows - folder for windows build with Intel Fortran and Visual Studio



For more information run: wwdisp.exe --help
